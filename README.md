# 图书管理系统 java863

## 项目概述

图书管理系统是一个集用户管理、图书管理、借阅管理等功能于一体的信息化平台。它通过优化图书管理流程，提高图书利用率，方便用户快捷地查询、借阅和归还图书。

## 技术栈

- **前端：** Vue3
- **后端：** Node.js

## 功能模块

### 用户管理

- 用户增删改查功能

### 首页

- 面板数据展示

### 图书管理

- 书籍的增删改查

### 借阅管理

- 借阅功能
- 归还功能
- 续租功能

## 系统角色

- 普通用户
